# Global Happiness Analysis
Socio-economic study of 158 countries to identify happiness drivers.

## Key Findings
- Job Satisfaction strongly correlates with happiness (r=0.80)
- Random Forest predicts happiness rank with 99.8% accuracy
- Economy (GDP) is the single strongest predictor of happiness score

## Tech Stack
Python, Pandas, Scikit-learn, Matplotlib, Seaborn

## How to Run
1. Clone the repo
2. Place world_happiness.csv in the same folder
3. Open and run the notebook
